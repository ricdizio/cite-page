SHANK

Este proyecto se desarrollo para la categoria de Laberinto de la competencia USBBOTS 2016. Este micromouse es capaz de resolver un laberinto de cualquier dimension con el algoritmo flood-fill.

Hardware

  • Microcontrolador:

    • TM4C123GH6PMI7

  • Motores:

    • 1624T003S

  • Driver Motores:

    • DRV8833PWPR

  • Gyro:

    • MPU-9250

  • Regulador

    • PTH04T261WAD

  • Baterias

    • 850 mAh standard battery

  • Ruedas:

    • KYOSHO MINI-Z
